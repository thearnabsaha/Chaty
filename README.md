# Chaty

Chaty is a real-time chat application built with React, TypeScript, and Vite. It allows users to create temporary chat rooms that expire after both users exit.

## Features

- Create and join temporary chat rooms.
- Light and dark theme toggle.
- Responsive and modern UI.

## Tech Stack

- **Frontend**: React, TypeScript, TailwindCSS
- **Build Tool**: Vite
- **Icons**: Lucide React, React Icons

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/chaty.git
   cd chaty
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open the app in your browser at `http://localhost:5173`.

## Scripts

- `dev`: Start the development server.
- `build`: Build the project for production.
- `lint`: Run ESLint to check for code issues.
- `preview`: Preview the production build.
